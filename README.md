# Quote and Booking Funnel Landing Page - Portfolio Sample

This project is a conversion-focused landing page sample built with Next.js and TypeScript.
It demonstrates how messaging, package framing, and quote capture can work together to improve booking intent.

## Case Study

### Problem

Many service landing pages attract traffic but lose high-intent visitors before they request a quote.

### Constraints

- Keep the page fast and lightweight
- Preserve clear CTA hierarchy on mobile and desktop
- Keep content easy to repurpose across business niches

### Solution

- Reframed hero and section sequence around quote-to-booking flow
- Tuned value proposition and features to reduce decision friction
- Added proof-style case-study metrics section on the homepage

### Architecture

- Next.js App Router
- TypeScript for maintainable component contracts
- Tailwind CSS for rapid responsive design

### QA

- File-level diagnostics after copy and section updates
- Local run verification for visual review

### Results (Illustrative Portfolio Benchmarks)

- +24% quote request completion
- -35% response delay before first callback
- +19% package page to booking intent conversion

### Tradeoffs

- Metrics are illustrative and not from a live production dataset
- Focus was on conversion messaging and UX flow over backend lead management

### Next Iteration

- Add lead source tracking and form analytics
- Integrate booking calendar API
- Add A/B variants for CTA copy and pricing card order

## Tech Stack

- Next.js
- React

- Tailwind CSS
